## Section 2 - CyberChef Challenges

### Mod 26 (ROT13)
- Challenge gave encoded text, had to decode using ROT13
- Tool: CyberChef → ROT13 operation
- Learned: ROT13 shifts every letter by 13 places, Caesar cipher variant

### Warmed Up (Hex to Decimal)
- Challenge: convert 0x3D (base 16) to decimal (base 10)
- Answer: 61 → submitted as picoCTF{61}
- Learned: 0x prefix means hexadecimal, remove it before converting

### 2warm (Decimal to Binary)
- Challenge: convert decimal number to binary
- Tool: CyberChef → To Base operation
- Learned: computers store everything in binary (base 2)

### Bases (Base64)
- Challenge: decode bDNhcm5fdGgzX3IwcDM1
- Tool: CyberChef → From Base64
- Output: l3arn_th3_r0p35
- Learned: Base64 strings look like random letters/numbers with no symbols
- Pattern recognition: if no special chars and random looking → try Base64 first
