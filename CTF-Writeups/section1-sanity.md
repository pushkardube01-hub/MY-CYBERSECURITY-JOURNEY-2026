## Section 1 - Sanity Challenges

### Obedient Cat
- Downloaded a file, opened in notepad, flag was inside

### Super SSH
- Connected to a remote server using SSH command
- Command: ssh ctf-player@titan.picoctf.net -p [port]
- Learned: SSH is used to securely access remote computers

### what's a net cat?
- Connected using netcat (nc) - simpler than SSH, no password
- Command: nc fickle-tempest.picoctf.net 53563
- Learned: netcat is used to read/write raw network connections
